```

```

## What YOU need to run (to complete the paper's results)

### Easy mode: one command (~60 min)

```bash
python3 run_full_paper_experiments.py
```

This is idempotent -- already-completed runs are skipped, so you can interrupt
(Ctrl-C) and resume at any time. It runs three phases in order:

  Phase 1: Finish main Nested-MARL seed 9  (~2 min)
  Phase 2: 3 ablations x 5 seeds at severity 1.5  (~30 min)
  Phase 3: Severity sweep {0.5, 1.0, 2.0} x 3 seeds x 2 methods  (~30 min)

### Manual mode (if you prefer to run phases separately)

```bash
# Phase 1: finish main
python3 train.py --method nested --episodes 80 --horizon 150 --seed 9 --severity 1.5 --log_every 80

# Phase 2: ablations
for seed in 0 1 2 3 4; do
  for m in nested_no_ema nested_no_timescale nested_two_level; do
    python3 train.py --method $m --episodes 80 --horizon 150 --seed $seed --severity 1.5 --log_every 80
  done
done

# Phase 3: severity sweep
for sev in 0.5 1.0 2.0; do
  for seed in 0 1 2; do
    python3 train.py --method ippo   --episodes 80 --horizon 150 --seed $seed --severity $sev --log_every 80
    python3 train.py --method nested --episodes 80 --horizon 150 --seed $seed --severity $sev --log_every 80
  done
done
```

import json, numpy as np
from pathlib import Path
from collections import defaultdict
d = defaultdict(dict)
for p in sorted(Path('results').glob('*.json')):
    data = json.load(open(p))
    m, s = data['args']['method'], data['args']['seed']
    sev = data['args'].get('severity', 1.0)
    d[(m, sev)][s] = data['metrics']['episode_reward']
for (m, sev), runs in sorted(d.items()):
    arr = np.array([runs[s] for s in sorted(runs)])
    print(f'{m:25s} sev={sev:.1f}: {arr.shape[0]} seeds, '
          f'mean-all={arr.mean():.2f}, first40={arr[:,:40].mean():.2f}, '
          f'last20={arr[:,-20:].mean():.2f}')
"

## Key hyperparameters (already tuned, don't change unless experimenting)

- `--severity 1.5` -- drift severity; 1.5 is where the gap is cleanest
- `--horizon 150` -- steps per episode; drift has ~3 regime shifts per episode
- `--episodes 80` -- long enough to see the full learning curve
- Nested-MARL rates: `alpha_l0=1e-4, alpha_l1=5e-4, alpha_l2=2e-3` (set inside `agents/nested_marl.py`)
- Drift: `medium_period=15, slow_period=50, continuous=True` (`drift/schedulers.py`)
